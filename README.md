# Modern University Landing Page

A premium cinematic university landing page built using **Next.js**, **React**, **TailwindCSS**, **GSAP**, and **Framer Motion**.

Inspired by modern editorial web experiences with immersive visuals, smooth interactions, responsive layouts, and performance-focused frontend architecture.  
The project uses JG University as the primary design and content reference.

---

## Tech Stack

- Next.js
- React
- TailwindCSS
- GSAP + ScrollTrigger
- Framer Motion
- Lucide React

---

## Features

- Cinematic fullscreen hero sections
- GSAP-powered scroll interactions
- Infinite horizontal auto-scroll galleries
- Touch-scrollable campus experience
- Fully responsive mobile, tablet, and desktop layouts
- Editorial-style UI design
- Responsive admissions & contact pages
- Optimized CTA sections
- Smooth animated navigation system
- Performance optimized image handling
- Mobile-first responsive architecture

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```

### Start Production Server

```bash
npm run start
```
